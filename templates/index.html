Inventory Tracker

Welcome to Inventory Tracker! This web application is meant for tracking household products like food and toiletries. This site can use a barcode scanner for improved efficiency. If you are able to set a custom prefix on your barcode scanner, we recommend setting it the ESC key. The source code for this web application can be found at https://github.com/Geodude25trade/InventorySystem.
